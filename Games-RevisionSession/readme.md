# System Requirements
GameZone is a small independent video game retailer that wants to digitize its inventory management system. Currently, all game stock is tracked manually, which leads to errors and inefficiencies. The company has asked you to build a simple web-based system using PHP and SQL to manage their game inventory. 

## Requirements
- Users can log into the system to view items within the store
- Admins can add, edit and delete items within the store
- Other users cannot access the system
- Games are displayed which includes game name, price and stock
