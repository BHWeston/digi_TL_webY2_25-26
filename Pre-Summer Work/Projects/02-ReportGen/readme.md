# Report Generator Task
You are asked to build a simple PHP script that simulates a student report generator. The script should:

## Question 1:
Create an inddexed array of at least 3 student names.

## Question 2:
Create a multidimensional array where each student has:
- A name (string)
- An associative array of 3 subjects and their corresponding marks

## Question 3:
Display each student’s name and their average mark.

## Question 4:
Use string functions to:
- Convert the student’s name to uppercase
- Trim any extra whitespace from subject names
- Replace any instance of the word "Math" with "Mathematics"
- Display the total number of characters in each subject name
